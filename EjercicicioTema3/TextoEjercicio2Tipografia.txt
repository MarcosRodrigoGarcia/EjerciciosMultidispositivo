Bootstrap incluye clases CSS para transformar la capitalización (mayúsculas/minúsculas):
Mostrar texto en minúsculas: "text-lowercase".
Mostrar texto en mayúsculas: "text-uppercase".
Mostrar texto con inicial en mayúsculas: "text-capitalize".

Para delimitar texto citado, podemos utilizar "blockquote" alrededor de cualquier contenido HTML. Usualmente es un párrafo delimitado con la etiqueta "p": 


En algún lugar de la mancha, de cuyo nombre no quiero acordarme...

También podemos hacer referencia a la fuente o autor de una cita con un "footer" dentro de la misma:


En algún lugar de la mancha, de cuyo nombre no quiero acordarme...

— Miguel de Cervantes en El Quijote.

Y por último, utilizar la clase "blockquote-reverse" para alinear la cita a la derecha:


En algún lugar de la mancha, de cuyo nombre no quiero acordarme...

Miguel de Cervantes en El Quijote. —

Para mostrar fragmentos de código, podemos utilizar:
•La etiqueta "code" para delimitar código "en línea": <section> (podemos continuar con texto).
•La etiqueta "pre" (y opcionalmente, seguida de "code") para bloques completos de texto: 
aptitude update
aptitude upgrade


•Si queremos representar algo que el usuario debe introducir en el teclado (como una combinación de teclas, por ejemplo), utilizamos "kbd": Ctrl + Alt + Supr 
•Manteniendo el principio semántico de HTML5, si queremos expresar variables (por ejemplo, como parte de una fórmula, utilizamos "var": c2 = a2 + b2 
•Y para delimitar ejemplos de salida de comandos o programas, utilizamos "samp": Estado actual: 2 actualizaciones [+2]. 

En caso de querer mostrar código que pueda incluir caracteres especiales para HTML, como etiquetas, signos de "mayor que" o "menor que", se recomienda utilizar la sintaxis de códigos o nombres de entidades.
